What is Jembe?

Python web framework for developing modern web apps,
build on top of Flask, designed for:

  • Rapid development using components, actions and events;
  • Easy creating truly reusable web components;
  • Developing without writing much code in JavaScript.
Is Jembe for You?

Yes, if You're:

  • Building web apps all by yourself;
  • Coding “same logic” between "similar pages"
    in backend and in frontend;
  • Already coding or don't mind coding in Python?

from jembe import Component
from ..model import Project

@page("projects")
class Projects(Component):

	def __init__(self, search:str=””):
        def super().__init__()

	def display(self):
		self.projects = Projects.query()
		if self.state.search:
			self.projects = self.projects.filter(
            Project.name.like=”{}%”.format(self.state.search)
            )
        return super().display()

How does Jembe works

  • On initial request Jembe renders the whole page, composed of one or multiple components;
  • When user interacts with any component, information of current state of all components and user interactions are send back to server via AJAX request;
  • Server executes actions associated with user interactions and re-renders only components that have been changed;
  • Browser receives HTML and mutates DOM according to changes in re-rendered components.
